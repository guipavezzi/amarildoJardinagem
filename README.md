# AmarildoJardinagem 🌿

Este é o repositório do projeto desenvolvido para **Amarildo Pavezzi**, profissional com mais de 20 anos de experiência em jardinagem e paisagismo em Campinas/SP. O objetivo do site é apresentar seus serviços, exibir trabalhos realizados e facilitar o contato com novos clientes.

## 🚀 Sobre o Projeto

O site funciona como uma vitrine digital, apresentando a trajetória do profissional e detalhando serviços como:
* **Manutenção e limpeza de jardins**
* **Projetos de paisagismo**
* **Cuidados periódicos e revitalização de áreas verdes**
* **Galeria de fotos com resultados de projetos reais**

> **Nota:** Este projeto foi desenvolvido como uma solução real para profissionais autônomos, unindo design moderno com performance para otimização de busca local.

## 🛠 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **Angular** (Framework Web)
* **TypeScript** (Lógica e Tipagem)
* **HTML5** & **Tailwind** (Estrutura e Estilização)

## 💻 Como rodar o projeto localmente

Siga os passos abaixo para configurar o ambiente e executar o site em sua máquina:

### 1. Clonar o repositório
```bash
git clone https://github.com/guipavezzi/amarildoJardinagem.git
cd amarildoJardinagem
```

### 2. Instalar as dependências
Certifique-se de ter o Node.js instalado.

```bash
npm install -g @angular/cli@21.2.0
```
```bash
npm install
```
3. Executar o servidor de desenvolvimento

```bash
ng serve
```
4. Acessar no navegador
Após o comando acima, o site estará disponível no endereço:
👉 http://localhost:4200
